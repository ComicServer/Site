Cesium is a jQuery-based UI with support for Bootstrap*. The ComicServer webapp (Cesium/Cornflower) is currently testing the UI and seeing what works. Expected release around Febuary 2022. The UI modernizes any jQueryUI/Bootstrap* site and creates an easy to edit experiance, with great design all the way.
