Important

This bot is for comic STRIPS only. Full list of comics is here. We also wanted to point out, when typing a comic name, type it as one word and with no special characters like & and $. An example is calvinandhobbes, one word and no special characters.

Definitions

codename - This is just the comic name. This needs to be one word.

prefix - cs!

Commands

help - All the commands!

rss - RSS for comic strips.

media - Comic media (videos).

comic [codename] - Today's daily comic!

search [codename] [query] - Search each comic (cs! search

search-global [query] - Search all of the comics on GoComics (cs!search-global )

site - A link to our site.

stats - ComicServer bot stats!

kingdom [codename] - Get comics from Comics Kingdom

version - Current bot version.

transcript [codename] [yyyy] [m] [d] - Get a comic's transcript!

date [codename] - Selected date from comic (yyyy m d).

random [codename] - Random comic for your choice!

ago [codename] [years ago] - Get a comic strip from a selected amount of years ago!
